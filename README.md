Task Manager App

A simple, browser-based task management app built with plain HTML, CSS, and JavaScript. No frameworks, no installs , just open and use.

How to Run the app

Option 1 - Run the app locally
    Download all three files into the same folder
        - index.html
        -style.css
        -app.js 

    Double click index.html to open it in any browser
        - note: all files must be in the same folder or the styles and logic will not load.

Option 2 - application is hosted. 
    link: https://task-manager-app-ashy-omega.vercel.app/


File structure
    task-manager/
     index.html → App structure and layout
     style.css  → Styling and visual design
     app.js     → All logic: add, complete, delete, save
